# CT Paper

CT Paper is a web platform created to access and manage class test papers for practice and preparation.  
It helps students find papers easily and revise in a simple, organized way.

## Live Demo
- Live View: https://skverma895746.github.io/RECB-All-CT-papers/

## Features
- Easy access to class test papers
- Simple and clean user interface
- Fast navigation between pages
- Helpful for exam preparation and practice

## Technologies Used
- HTML
- CSS
- JavaScript
- google drive for storage

## Project Description
This project is designed to make class test papers available in one place so students can practice more efficiently.  
The interface is lightweight, responsive, and easy to use on different screen sizes.
